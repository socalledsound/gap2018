FINAL PROJECT

Your final project should be the culmination of this semester.

It may be the continuation and completion of a game that you've been working on all semester or it may be a game which you're just starting.

Or it may be an artwork in your favorite medium which works with the ideas that we've been talking about all semester.

If it's the game you've been working on all semester, now is the time to refine the user experience and do all the things that you've been putting off until now.

Think more carefully about the user interface and the user experience.

If it's a new game or artwork that you're just starting, refer to the links below for inspiration. These are works of art that I think are fun and interactive in engaging ways.

The most important thing is that it be something that you love, are passionate about, something that you're proud of.

Let me know what you're planning to do ASAP so I can help you think about it and help you implement what you're planning to do.

And, please, dont forget to have fun! The best things come from fun.
